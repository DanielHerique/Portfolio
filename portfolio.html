Projeto 1 - Criação de Conteúdo

Este projeto consistiu na criação de conteúdo personalizado para diferentes tipos de empreendedores. O objetivo foi entender as necessidades de cada segmento e
desenvolver postagens adaptadas ao seu público-alvo, utilizando estratégias de marketing digital e produção de conteúdo relevante.

Objetivo: Ajudar pequenos negócios e empreendedores a se destacarem nas redes sociais com conteúdos autênticos e direcionados.

Projeto 2 - Desenvolvimento de Site Responsivo

O Projeto 2 focou na criação de um site responsivo para uma pequena empresa, com o objetivo de melhorar sua presença online. O site foi desenvolvido utilizando as melhores práticas de design e usabilidade, garantindo uma navegação fluida em dispositivos móveis e desktop.

Objetivo: Proporcionar uma experiência de usuário consistente em todos os dispositivos, melhorando a conversão e o engajamento do público.

Projeto 3 - Campanha de Mídias Pagas

Para o Projeto 3, criamos uma campanha de anúncios pagos para um cliente no Facebook e Instagram. A campanha foi segmentada com base em dados demográficos e comportamentais para alcançar um público específico e aumentar as vendas.

Objetivo: Aumentar as conversões e a visibilidade do cliente nas plataformas digitais, otimizando os investimentos em mídia paga.

Projeto 4 - VideoMaker

O Projeto 4 é um espaço de criação de conteúdo visual, onde desenvolvo vídeos personalizados para redes sociais. Ofereço serviços de criação, gravação e edição para eventos, publicidade, documentários e conteúdo digital, utilizando software de edição avançado, equipamentos de alta qualidade e criatividade para entregar resultados impactantes.

Serviços
1. Gravação de eventos: Registro de casamentos, festas, conferências e outros eventos importantes.
2. Edição de vídeos: Criação de vídeos editados a partir de gravações próprias ou recebidas.
3. Pós-produção: Adição de efeitos, transições, áudio e legendas para realçar a narrativa.
4. Criação de conteúdo: Desenvolvimento de vídeos institucionais, publicitários e de divulgação.
