Meus Conhecimentos em Tecnologia

Na faculdade, venho adquirindo uma base sólida em várias áreas de tecnologia, o que está ampliando minha visão e habilidades no desenvolvimento de sistemas e soluções digitais.

Em Lógica de Programação, aprendi a pensar de forma estruturada para resolver problemas, entendendo a sequência de passos que o computador deve seguir. Usei a linguagem C para praticar a lógica de programação, o que me ensinou a desenvolver algoritmos eficientes e funcionais.

Em Desenvolvimento Web Básico, estou aprendendo a criar páginas web com HTML, CSS e JavaScript. O HTML permite estruturar o conteúdo, o CSS estiliza e dá forma à aparência das páginas, enquanto o JavaScript traz interatividade, tornando as páginas mais dinâmicas e envolventes para os usuários.

Mais recentemente, iniciei meus estudos em Banco de Dados, entendendo como organizar e gerenciar informações de forma eficiente e segura. Esse conhecimento é essencial para criar sistemas que possam armazenar e recuperar dados, como informações de clientes ou produtos, de maneira prática e funcional.
